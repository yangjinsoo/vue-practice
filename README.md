# practice-vue
- Vue.js 학습용 프로젝트

## Project setup
```
npm install -g @vue/cli
vue.cmd create 프로젝트명
```
- 노드 설치 (npm이용하기 위함)
- vue 설치
- vue create로 프로젝트 생성
    - 보안 오류 뜨면 vue.cmd create
